Документация API генерации видео с аватарами

Эта документация описывает два различных генератора видео:

  • Генератор 1: Генерирует видео с аватаром ann и поддерживает настройку голоса
  • Генератор 2: Поддерживает более широкий выбор аватаров (irina, ann, malyshko, ksenija)

Перед использованием API необходимо:

  1. Заменить значение serviceId на актуальный ID сервиса
  2. Использовать действительный ключ авторизации в заголовке запроса
